border-box;"><section class="" style="text-align: left;font-size: 14px;box-sizing: border-box;"><p style="box-sizing: border-box;">尊敬的钟志华校长：</p><p style="text-indent: 2em;box-sizing: border-box;">您好！</p><p style="text-indent: 2em;box-sizing: border-box;">我们是同济大学的学生。近期一系列高校性骚扰事件引起了社会的广泛关注，北京航空航天大学已毕业博士罗茜茜实名举报博士导师陈小武长期性骚扰多名学生的事件更引起公众的广泛讨论。这不禁让人回想起2014年厦门大学的性骚扰事件，那之后，教育部将对学生实施性骚扰或与学生发生不正当关系列为师德 “红七条” 之一。可事实证明，通过师德建设处理高校性骚扰问题还远远不够。根据媒体NGOCN的统计，从2014年到2017年四年内曝出的13起高校教师性骚扰事件，1/3查无后续。更有广州性别教育中心在2017年4月发布的《中国大学生在校和毕业生遭遇性骚扰状况调查》表明，近七成受访者遭受过不同形式的性骚扰，女性遭受性骚扰的比例为75%，男性为36%。</p><p style="text-indent: 2em;box-sizing: border-box;"><br style="box-sizing: border-box;"  /></p><p style="box-sizing: border-box;">&nbsp; &nbsp; &nbsp; &nbsp;中国高校性骚扰事件的频发令人震惊，因此更值得我们关注和深思，究竟是什么让性骚扰屡禁不绝，是什么让被骚扰者有苦难诉？首先需要明确的是，性骚扰的本质不止关乎性，性只是一个载体，它实际上侵犯了被骚扰者的人格尊严。可性教育的缺失使得被骚扰者往往难以启齿，从心理到行为都不能很好地处理性骚扰，有时即使站出来也得不到公义。机制的缺位则让受害者无处申诉，难以得到有效帮&nbsp; &nbsp; 助。</p><p style="box-sizing: border-box;"><br style="box-sizing: border-box;"  /></p><p style="box-sizing: border-box;">&nbsp; &nbsp; &nbsp; &nbsp; 高等教育性骚扰事件存在着特殊现象，即在学术和招聘中将性作为交换条件，或者造成敌视、胁迫或侵犯性的学术和工作环境的不当行为。在教育项目或活动中加害人使用语言、非语言和身体方面的性行为对受害人施加持续广泛的不良影响。可见校园性骚扰的源头并非只是骚扰者的道德问题，反校园性骚扰绝非依靠道德建设就能解决。没有防治性骚扰的有效机制保障被骚扰者的权益，从社交媒体发声，形成倒逼压力，往往成了没有办法的办法。不仅让受害者身心受损，也给学校、教职工及学生带来极大的负面影响，相信任何人都不会愿意走上这一步。如果能下狠心落实性骚扰防治体系的建设，即便有事件被曝光，学校在之后的处理过程中也能高效公正的解决事端。比起“曝光一事解决一事”的态度和做法，合理有效的机制才能防范于未然。</p><p style="box-sizing: border-box;">&nbsp; &nbsp; &nbsp; &nbsp;北航事件曝光后，教育部不仅表示对高校性骚扰零容忍，还明确表态将着手同有关部门认真研究建立健全高校预防性骚扰的长效机制。我们也真诚地希望在建设世界一流学府的路上，同济大学能正视和关注校园性骚扰防治，建立防治性骚扰机制。为此，我们提出以下几点建议：</p><p style="box-sizing: border-box;">&nbsp; &nbsp; &nbsp; &nbsp;1.&nbsp;&nbsp;&nbsp;&nbsp; 开设适合大学生阶段的、全面的性教育课程，性教育是根本的解决途径，也是学校责任所系，将防治性骚扰教育纳入师生日常教学和学习的一个模块。对教师、科研教学的辅助人员、行政人员甚至后勤人员开展防治性骚扰的培训并在每一学年末开展相应考核，课程培训可与杨浦区相关社工组织合作展开；</p><p style="box-sizing: border-box;">&nbsp; &nbsp; &nbsp; 2.&nbsp;&nbsp;&nbsp; 组织各种相关讲座、纪录片放映，艺术创作，工作坊和讨论会以及各种手段开展防治性骚扰的教育；</p><p style="box-sizing: border-box;">&nbsp; &nbsp; &nbsp; 3.&nbsp;&nbsp;&nbsp;&nbsp; 制定针对性骚扰或基于性别暴力的指导手册，在校内张贴反性骚扰海报，并明确注明相关概念定义、求助资源和处理方法；</p><p style="box-sizing: border-box;">&nbsp; &nbsp; &nbsp; 4.&nbsp;&nbsp;&nbsp;&nbsp; 加强治安管理，在校园偏僻处增加路灯，设立保安亭，加强巡逻等；</p><p style="box-sizing: border-box;">&nbsp; &nbsp; &nbsp; 5.&nbsp;&nbsp;&nbsp;&nbsp; 建立专门的防治性骚扰办公室负责相关事件，并有明确的负责人和联系人，设立反性骚扰热线；</p><p style="box-sizing: border-box;">&nbsp; &nbsp; &nbsp; 6.&nbsp;&nbsp;&nbsp;&nbsp; 每学期展开一次性骚扰的网络调查，让师生可以对性骚扰、抑郁、焦虑等情况进行在线匿名反馈；</p><p style="box-sizing: border-box;">&nbsp; &nbsp; &nbsp; 7.&nbsp;&nbsp;&nbsp;&nbsp; 保存举报及调查证据，在处理情节严重事件时（例如尾随、强奸、强奸未遂）与当地警方合作执行；</p><p style="box-sizing: border-box;">&nbsp; &nbsp; &nbsp; 8.&nbsp;&nbsp;&nbsp;&nbsp; 在所有防治与反应过程中，学生（教职工）代表需与受害学生（教职工）以相同立场全程参与；</p><p style="box-sizing: border-box;">&nbsp; &nbsp; &nbsp; 9.&nbsp;&nbsp;&nbsp;&nbsp; 针对被正式举报的骚扰行为，校方应在公开的期限内（比如，20天）内给出初步答复，答复须对事件发生原委与严重程度给予初步判断；</p><p style="box-sizing: border-box;">&nbsp; &nbsp; &nbsp;10.&nbsp; &nbsp; &nbsp;对举报信息、处理过程都采取最严肃的匿名和信息保存机制；</p><p style="box-sizing:
